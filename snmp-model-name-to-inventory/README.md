# SNMP Model and Name to Inventory
#### SNMP Model and Name to Inventory

#### Authors:
- sergio.cricca@tomware.it

#### Features:
Script checks if device name and device model are present and copies its content into inventory.
This is due to the unavailability of multiple items filling in the same field.


#### Usage:
```snmp-info-to-inventory.py``` -s {HOST.NAME}
